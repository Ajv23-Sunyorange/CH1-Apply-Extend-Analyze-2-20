Guideline 1.1 provides text alternatives for people who aren't able to read traditional text like braille, symbols or simpler vocab.

Guideline 1.2 provides an alternative form of audio/video prerecorded videos like added subtitles or text to speech.

Guideline 1.3 creates content to be presented in many different ways, like choosing a more meaningful sequence but still not losing any information.

Guideline 1.4 shows you how to make it easier for the user to see and hear content by using color, audio control, text size control and other methods to seperate it from the background.
